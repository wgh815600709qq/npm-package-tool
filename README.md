# npm-package-tool
一个常用工具函数整理后的npm工具包


## 当前版本v1.0.1

```

    npm install wgh_tool
    
```

##  一些扩展功能
    集成压缩
    ``` 
    
        npm run build
        
    ```
    
    集成全部函数单元测试、
    ``` 
    
        npm run test-all
        
    ```
    
    部分函数单元测试
    
    ```
    
        1、将需要测试的函数名写入unit-testing/test-task.js数组内
        
        2、npm run test-task
        
    ```
